# BookMass Searcher v4

A blazing-fast Chrome extension to **search dozens of books instantly** on Anna’s Archive, The Pirate Bay, LibGen, Ocean of PDF, Z-Library, Mobilism and more — all opened in a **perfectly colored tab group** with one click.

## Why you’ll love it

-   Paste 5 or 150 book titles → one click → everything opens grouped
-   Your favorite sites always on top
-   Reuses existing groups (no duplicates)
-   100 % offline · no tracking · works forever

> **Warning: Some sites are unofficial mirrors. Use responsibly and according to your local laws.**

## Installation in 30 seconds (2025 method)

```bash
# 1. Clone the repo (or download ZIP and extract)
git clone https://github.com/eyvr17/DWBooks.git
# or just download the ZIP from the repo and extract it
```

# 2. Enter the folder

`cd DWBooks/DWBooks-v4`

# 3. Open Chrome → chrome://extensions/

```bash
#    → Turn on "Developer mode" (top right)
#    → Click "Load unpacked"
#    → Select the DWBooks folder you just downloaded
#    → Done! The icon appears instantly
```

**Enjoy immediately — no build, no extra steps!**

### Quick Usage

1. Click the extension icon
2. Paste your book list (one title per line)
3. Anna’s Archive is selected by default (or pick any other)
4. (Optional) Change group name & color
5. Click **OPEN ALL TABS**
6. Watch the magic: all searches open in a beautiful colored group
